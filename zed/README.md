# Zed Editor

Test with [Zed](https://zed.dev/) [0.137.6] or later.

## Requirements

### Fonts

* Fira Code
* IBM Plex Mono
* IBM Plex Sans
* JetBrains Mono
* MesloLG Nerd Font
* Monaspace

### Formatter

* black (Python)
* prettier (HTML, JavaScript, TypeScript)
* clippy (Rust)

### Theme

* Tokyo Night
